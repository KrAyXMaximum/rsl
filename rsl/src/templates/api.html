API Documentation

GET /api/status

Shows the latest status data available.
Response Schema:

{
    "services": [
        {
            "code": 200,
            "guess": ["up", "No problems detected"],
            "id": "auth",
            "name": "Authentication",
            "ping": 115.52,
            "url": "https://auth.roblox.com/"
        },
        ...
    ],
    "status": ["online", "green"]
}

Example URL: https://robloxstatus.live/api/status

GET /api/historical

Retrieves the past 24 hours of historical data for a given API.
Response Schema:

{
    "data": {
        "02:10": 451.34,
        "02:11": 198.39,
        "02:12": 186.87,
        ...
    },
    "name": "Authentication",
    "uptime": 100.0
}

Required parameters:

  • id - String - The service's ID

Example URL: https://robloxstatus.live/api/historical?id=auth
